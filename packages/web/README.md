## Public Website (x2)

## Prerequisites

1. Install [Node.js](https://nodejs.org/en/)
2. Install PNPM: `npm install -g pnpm`

## Installation

1. Run: `pnpm install`

## Development

1. Run: `pnpm run dev`

## Docker

`docker build -t public-website-x2 .`

`docker run -p 3000:3000 public-website-x2`
